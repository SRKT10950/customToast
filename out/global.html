showToast(title, message, variant, mode)

This function takes in a title, message, variant, and mode. If the mode is false, the toast will close after the duration
Parameters:
Name Type Description
title String The title of the toast
message string The message you want to display in the toast.
variant string This is the type of toast you want to show. It can be 'success', 'error', 'warning', 'info'.
mode boolean If you want to show the toast bar for a long time, you can set this to true.
